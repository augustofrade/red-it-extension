Red-It Settings

Quick Settings

Blocking Mode

The mode determines how content is blocked on the platform.

Hover over each option to see a description.

Content Blocking

Keywords Blocklist

User content is blocked based on the keywords specified in the blocklist.

Write the keywords you want to block, separated by line breaks.

Note: Use * to match any number of characters. Example: "red*" blocks posts with both "reddit" and "red-it" in their title.

Communities Blocklist

Write down the communities you want to block, separated by line breaks.

Note: Write the subreddit name without the "r/" prefix. Example: "funny" blocks posts from r/funny.

Comments Blocking

Comments are blocked based on the keywords specified in the blocklist.

Block Premium Ads

Enable this setting to block premium ads found on the platform.

Block NSFW Posts

Enable this setting to block posts that are marked as NSFW.

Technical

Log Usage into Developer Console

Enable this option to log all blocked content into the developer console.

Danger Zone

Delete Data

Delete all user data stored by the extension.

Reset Metrics

Reset all metrics collected by the extension.

Metrics are limited to the amount of subreddits and posts blocked by the extension.
